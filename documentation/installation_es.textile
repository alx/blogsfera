h1. Instalación en el servidor:

h2. Requerimientos

* Base de datos
* Apache, o cualquier servidor HTTP
* rewrite mod habilitado en servidor HTTP
* PHP

h2. Instalación (5 minutos)

* Descargar el código fuente1
* Suba el código al servidor seleccionado
* Llame al sitio web desde el navegador, encontrará el proceso de configuración
* Si es necesario, cree el wp-config.php en su directorio de instalación, y complete la configuración de la base de datos:

@@@
cp wp-config-sample.php wp-config.php
@@@

h2. Portal de la Blogsfera

* En Site Admin > Opciones, active el menu de plugins al final de la página
* En Plugins (menú derecho) active los plugins: Wordspew, Blogsfera Portal Widgets 
* Vuelva a Site Admin>Opciones, y desactive el menú de plugins
* Cree una nueva página. En Page Template escoja "Search_results". Pulse botón Publicar.
* Cree una nueva página. En Page Template escoja "Tag_results". Pulse botón Publicar. 

h2. Portal Feevy

* Cree un nuevo blog
* Active el tema feevy-portal en este blog
* Vaya a Site Admin>Feevy y configure las opciones en la página de administración (seguir instrucciones para la configuración).

h2. Comunidad

* Cree un nuevo blog
* Active el tema WP Contact Manager en este blog
