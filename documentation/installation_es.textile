h1. Instalación en el servidor

h2. Requerimientos

* PHP versión 4.3 o más reciente
* MySQL versión 4.0 o más reciente
* Para activar los permalinks, la extensión mod_rewrite para Apache es necesaria
* "Más acerca de los requerimientos Wordpress":http://wordpress.org/about/requirements/

h2. Instalación (5 minutos)

* "Descargar el código":http://github.com/alx/blogsfera/tarball/master
* Suba el código al servidor seleccionado
* Llame al sitio web desde el navegador, encontrará el proceso de configuración
* Si es necesario, cree el _wp-config.php_ en su directorio de instalación, y complete la configuración de la base de datos:

@@@
cp wp-config-sample.php wp-config.php
@@@

h1. Instalación de los portales

h2. Portal Feevy

* Cree un nuevo blog
* Active el tema _feevy-portal_ en este blog
* Vaya a _Site Admin>Feevy_ y configure las opciones en la página de administración (seguir instrucciones para la configuración).

h2. Comunidad

* Cree un nuevo blog
* Active el tema _WP Contact Manager_ en este blog
